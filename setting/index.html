Firewall settings

  1. Open up, Start -> Control panel -> Windows firewall, and choose "Allow a program or featureâ€¦"
  2. Mark all 3 checkbox under fw_upgrader and press "OK" (Login may be required under "Change settings")

Protocol settings

  1. Open up, Start -> Control panel -> Administrative Tools -> Windows firewall with Advanced Security
  2. Click -> Inbound Rules -> New Rule
  3. Choose custom and click next
  4. Enter "Protocol and Ports" and set the Protocol type to ICMPv4 and click customize
  5. Choose Specific ICMP types, check mark Echo Request, click OK and click nex
  6. Choose Any IP address in both and click next
  7. Choose Allow the connection and click next
  8. Check mark all three and click next
  9. Type in a name e.g. "Echo Request" and click finish

Port settings

Now the first rule is ready and the second rule can be configured

  1. Open up, Start -> Control panel -> Administrative Tools -> Windows firewall with Advanced Security
  2. Click -> Inbound Rules -> New Rule
  3. Choose port and click next
  4. Choose UDP and Specific local ports. Set it to 69 and click next
  5. Choose allow the connection and click next
  6. Check mark all three and click next
  7. Type in a name e.g. "TFTP UDP 69" and click finish
